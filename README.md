A full-stack beauty salon booking platform enabling seamless service browsing, booking, and approvals.

## Features
- User & Parlour authentication  
- Browse salons, services, and availability  
- Bookings with admin approvals via admin dashboards
- Notification system  
- Search & filtering UI to view neary options
- Protected routes for admins and users

## Project Highlights
- Built a web application to connect local beauty salons with customers.  
- Designed and implemented separate **admin** and **user dashboards** for efficient management and interaction.  
- Implemented a feature for users to **search and find nearby beauty salons** based on their location.  
- Developed a **booking system** enabling users to create, update, and delete appointments for home or onsite services.  

## Tech Stack
- **Frontend:** React, React Router, Web Vitals  
- **Backend:** Node.js, Express, CORS  
- **Database:** MongoDB Atlas  
- **Hosting / Infra:** GCP (self-service platform description), Cloud features  
- **Other:** Automated workflows, load testing, CI/CD ready  

## How to Run

### Frontend
```bash
cd client
npm install
npm start
```

### Backend
```bash
cd server
npm install
node server.js
```
